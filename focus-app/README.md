# FocusFlow

A minimal, privacy-first focus and productivity web app.

## Features
- Pomodoro-style timer with customizable durations
- Task list with estimates, tomato count, and completion
- LocalStorage persistence
- 7-day focus minutes chart (Chart.js)

## Usage
Open `index.html` in a modern browser. No backend.

## Roadmap
- Notifications and sound on session end
- PWA installability and offline support
- Keyboard shortcuts
- Export/import of data
